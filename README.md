# Truth or Dare AI — marketing site

Marketing + content site for the Truth or Dare AI iOS app. Next.js (App Router),
static export, Tailwind CSS. Deploys as plain static files to truthordareai.app.

## Develop

```bash
npm install
npm run dev
```

## Build (static export)

```bash
npm run build
```

Output goes to `out/`. That directory is the whole deployable site, upload it
to any static host (Vercel, Netlify, S3 + CloudFront, GitHub Pages, etc).

## Structure

- `src/app/` — routes. `/`, `/play`, and a dynamic `[category]` route that
  generates the six SEO question-hub pages at build time (see
  `src/lib/site-config.ts` for the category list).
- `src/components/` — `QuestionCard`, `AppStoreBadge`, `IntensitySlider`,
  `CategoryGrid`, plus the `/play` and home demo widgets.
- `src/lib/questions.ts` — the only place that reads `data/questions.json`.
  Category pages and the `/play` widget both go through this file.
- `data/` — the question bank. See `data/README.md` before touching it; it's
  generated by `scripts/build-questions.mjs` from `data/source/*.json`.

## Updating the App Store link

`APP_STORE_URL` in `src/lib/site-config.ts` is a placeholder until the app is
live. Update it there, nothing else needs to change.
